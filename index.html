Custom Serialization - `BaseXYZ` is `XYZ` from the previous slide ```python class Encodable: '''Empty base class to identify encodable classes.''' pass class Experiment(BaseExperiment, Encodable): _enc = ['name', 'details'] class DetailsTxt(BaseDetailsTxt, Encodable): _enc = ['text'] class DetailsNum(BaseDetailsNum, Encodable): _enc = ['number'] ``` - Don't worry, we'll get rid of `_enc` --- ## Custom serialization ```python JSON_CLS = '_json_cls' class Encoder(json.JSONEncoder): '''Custom JSON encoder.''' def default(self, obj): '''Record encodable objects as dicts with class name.''' if isinstance(obj, Encodable): class_name = obj.__class__.__name__ obj = {key:obj.__dict__[key] for key in obj._enc} obj[JSON_CLS] = class_name return obj ``` --- ## Custom serialization ```python tests = [ Experiment('with dictionary', {'k': 0}), Experiment('with text', DetailsTxt('the text')), Experiment('with number', DetailsNum(1234)) ] show('encodable test cases', tests) show('encoded JSON persistence', [json.dumps(e, cls=Encoder) for e in tests]) ``` ```console encodable test cases .. .. > .. > encoded JSON persistence .. {"name": "with dictionary", "details": {"k": 0}, "_json_cls": "Experiment"} .. {"name": "with text", "details": {"text": "the text", "_json_cls": "DetailsTxt"}, "_json_cls": "Experiment"} .. {"name": "with number", "details": {"number": 1234, "_json_cls": "DetailsNum"}, "_json_cls": "Experiment"} ``` --- ## Using Pydantic - [Pydantic][pydantic] is a data parsing and validation library ```python from pydantic import BaseModel, ValidationError class Encoder(json.JSONEncoder): def default(self, obj): * if isinstance(obj, BaseModel): * class_name = obj.__class__.__name__ * obj = obj.dict() * obj[JSON_CLS] = class_name elif isinstance(obj, SqlBase): columns = obj.__table__.columns.keys() class_name = obj.__class__.__name__ obj = {key:getattr(obj, key) for key in columns} obj[JSON_CLS] = class_name else: return json.JSONEncoder.default(self, obj) return obj ``` --- ## Using Pydantic ```python class ExperimentSql(SqlBase): __tablename__ = 'experiment' name = Column(String(255), primary_key=True, nullable=False) details = Column(JSONColumn()) class DetailsTxt(BaseModel): text: str class DetailsNum(BaseModel): number: int try: DetailsNum(number='not a number') except ValidationError as e: print('trying to create invalid Pydantic field:', e) ``` ```console trying to create invalid Pydantic field: 1 validation error for DetailsNum number value is not a valid integer (type=type_error.integer) ``` --- ##
